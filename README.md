# 🚀 Start：我的 C 语言学习仓库

这是我在 GitHub 上的第一个仓库。我会在这里记录学习 C 语言和数据结构的过程，并逐渐把这个仓库丰富起来。

---

## 📂 项目结构

- `c/` ：存放我的 C 语言代码
- `.gitignore` ：忽略规则配置（防止 .exe 等临时文件上传）
- `README.md` ：仓库说明书（也就是你现在看到的这个文件）
- `commit.bat` ：**一键上传更新**。双击即可将本地代码同步到 GitHub
- `pull.bat` ：**一键下载最新代码**。双击即可将 GitHub 仓库内容同步到本地

---

## 📝 日常使用指南

### 1. 上传代码（本地 -> GitHub）

> 写好代码后，**双击 `commit.bat`**，输入提交信息，即可完成上传。

### 2. 下载更新（GitHub -> 本地）

> 如果你想在其他电脑上继续写代码，**双击 `pull.bat`**，即可获取仓库里的最新版本。

### 3. 编译与运行（C 语言部分）

如果安装了 MinGW 环境，进入 `c` 文件夹，输入：

```bash
gcc first.c -o first
first.exe
```
